## Becoming a Maintainer

Maintainership is currently invitation-only as the project is founder-led. As the contributor base grows (Stage 2), the criteria and process for adding maintainers will be documented here. Maintainers commit to:

- Reviewing PRs within 7 days
- Participating in the monthly maintainer sync
- Following the Code of Conduct enforcement process
