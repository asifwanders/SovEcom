# Welcome to SovEcom

**SovEcom** is an open-source, headless ecommerce platform built for the European Union — from day one.

<CardGrid stagger>
  <Card title="EU-First" icon="seti:eu">
    RGPD-by-default, EU VAT correctness, and data sovereignty built into the core.
  </Card>
  <Card title="Headless" icon="seti:react">
    Separate frontend and backend. Use our reference storefront or build your own.
  </Card>
  <Card title="Open Source" icon="seti:git">
    AGPL-3.0 licensed core. No vendor lock-in, no feature gates.
  </Card>
  <Card title="Modern Stack" icon="seti:typescript">
    TypeScript, NestJS, PostgreSQL, Redis, and Meilisearch.
  </Card>
</CardGrid>
