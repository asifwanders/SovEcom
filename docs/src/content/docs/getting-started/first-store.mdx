# First Store

After starting the development environment, open the setup wizard at `http://localhost:5174` and follow the steps:

1. Enter the setup token (printed in Docker logs)
2. Configure brand settings
3. Set up database connection
4. Configure SMTP for emails
5. Set up payment providers
6. Configure tax settings
7. Select a theme
8. Create admin account
