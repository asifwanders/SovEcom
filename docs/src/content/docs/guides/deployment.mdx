# Deployment

## Docker Compose

The simplest production deployment:

```bash
docker compose up -d
```

## Environment Variables

Copy `.env.example` to `.env` and configure:

- `DATABASE_URL`
- `REDIS_URL`
- `MEILISEARCH_URL`
- `MEILI_MASTER_KEY`
