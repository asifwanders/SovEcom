# Multi-tenancy

SovEcom v1.0 is single-tenant, but every table includes a `tenant_id` column to support a future multi-tenant cloud deployment.

## Tenant Isolation

- Every query scopes by `tenant_id`
- Cache keys include tenant prefix
- Search indexes are tenant-scoped
